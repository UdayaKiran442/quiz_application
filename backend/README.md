# To run backend project

## Clone github repository
```sh
git clone https://github.com/UdayaKiran442/quiz_application
```

## Install dependencies
```sh
cd backend
```

```sh
bun install
```

## Run server

### For development environment
```sh
bun run dev
```

### For production environment
```sh
bun run prod
```
